# 医疗数据分析可视化实时监控系统 python37

## 项目概述

本系统是基于Python语言的医疗数据分析与可视化实时监控系统。通过前后端分离的设计模式，结合MySQL数据库、Flask框架、Vue框架以及机器学习算法，实现医疗数据的处理、预测和可视化功能。

## 技术栈

- 后端：Python语言、Flask框架
- 前端：Vue框架
- 数据库：MySQL
- 机器学习算法：随机森林分类算法

## 功能模块

### 系统架构

- **后端**：负责数据处理、算法训练和模型预测等核心功能。
- **前端**：提供用户交互界面和数据可视化展示。

### 数据处理与机器学习

- 数据获取：从MySQL数据库获取医辽文本数据。
- 数据预处理：包括文本清洗、分词、特征提取等步骤。
- 模型训练：利用随机森林分类算法对预处理后的数据进行训练。

### 病预测与可视化

- 文本数据预测：利用训练好的模型对新的文本数据进行类别预测。
- 数据展示：通过前端界面展示各类疾病的发病率、地域分布、时间趋势等信息。

## 系统角色

- 医疗工作者：使用系统进行数据分析和病预测。

## 运行环境

- 后端：Python环境、Flask框架
- 前端：Vue.js
- 数据库：MySQL数据库

## 部署步骤

1. 搭建Python环境。
2. 安装Flask框架。
3. 配置MySQL数据库。
4. 编译前端Vue项目。
5. 部署后端服务。
6. 部署前端页面。
